## 功能模块

系统的主要功能模块如下：

1. 语音录制与上传
   - 用户可录制WMV格式的语音并上传至系统。
2. 语音识别与分析
   - 采用自然语言处理方式分析语音内容。
   - 识别语音中的垃圾分类信息。
   - 支持普通话语音识别。
3. 垃圾分类反馈
   - 根据识别结果，提示用户相应的垃圾分类操作，如垃圾桶盖的自动打开。
